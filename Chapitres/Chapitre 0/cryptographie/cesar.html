Le Code César

En cryptologie, le chiffrement par décalage est connu sous le nom de code César: une méthode simple de chiffrement de messages utilisée par Jules César.
Le code César fonctionne en remplaçant chaque lettre du texte en clair par une lettre à distance fixe, toujours du même côté, dans l'ordre alphabétique. Pour les dernières lettres (dans le cas d'un décalage à droite), on reprend au début. Par exemple avec un décalage de 3 vers la droite, A est remplacé par D, B devient E, et ainsi jusqu'à W qui devient Z, puis X devient A.
