Les Différents Types de Langages de Programmation

Il existe de nombreux langues de programmations qui peu être trier par type de fonctionnement :

  • Langages Compilés (C, C++)
    Les langages compilés transforment le code source en code machine, avant d'être excuter.
  • Langages Interprétés (Python)
    Les langages interprétés exécutent le code ligne par ligne.
  • Langages Mixtes (Java).
    Les langages mixtes combinent les deux approches.
  • Langages Orientés Objet : (Java)
    Basés sur le concept d'objets.
    Ces langages organisent le code en objets qui contiennent des données et des méthodes, facilitant la réutilisation et la modularité.
  • Langages de bas niveau : (assembleur)
    Proches du matériel
    Ils permettent un contrôle direct du matériel et sont souvent utilisés pour des applications nécessitant une performance maximale.
  • Langages de haut niveau : (Python)
    Plus abstraits et faciles à utiliser
    Ils offrent une syntaxe plus proche du langage humain, ce qui facilite le développement et la compréhension du code.
